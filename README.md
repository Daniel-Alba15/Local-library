
# Local library

A basic CRUD library application usinga Node.js, MongoDB and Pug.js.

## Run Locally

Clone the project

```bash
  git clone https://github.com/Daniel-Alba15/Local-library.git
```

Go to the project directory

```bash
  cd Local-library
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
